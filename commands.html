Command List

コマンド集

TSUNAMI SMPで利用できる主要コマンド一覧です。
サバイバル、経済、EliteMobs、ギルドなど、よく使うコマンドをまとめています。

コマンド検索

```

コマンド名や説明文で検索できます。

```
```

基本・案内

/help

利用できるコマンドのヘルプを確認します。

/rules

サーバールールを確認します。

/spawn

スポーン地点へ移動します。

/list

現在オンラインのプレイヤーを確認します。

```
```

移動・テレポート

/sethome

現在地をホーム地点として設定します。

/home

設定したホームへ移動します。

/delhome

設定したホームを削除します。

/tpa 名前

指定したプレイヤーの場所へテレポート申請を送ります。

/tpahere 名前

指定したプレイヤーを自分の場所へ呼ぶ申請を送ります。

/tpaccept

届いたテレポート申請を承認します。

/tpdeny

届いたテレポート申請を拒否します。

/rtp

ランダムな場所へテレポートします。

/tpr

ランダムテレポートを行います。サーバー設定によりクールダウンがあります。

```
```

経済・ショップ

/bal

自分の所持金を確認します。

/balance

所持金を確認します。/bal と同じ用途です。

/pay 名前 金額

指定したプレイヤーへ送金します。

/baltop

サーバー内の資産ランキングを確認します。

/shop

サーバーショップを開き、アイテムの購入や売却を行います。

/sellgui

売却用GUIを開き、対応アイテムをまとめて売却します。

/sellall

売却可能なアイテムをまとめて売却します。

```
```

チャット・交流

/msg 名前 内容

指定したプレイヤーへ個人メッセージを送信します。

/tell 名前 内容

個人メッセージを送信します。/msg と同じ用途です。

/reply 内容

最後にメッセージを送った相手、または受け取った相手へ返信します。

/r 内容

/reply の短縮形です。

/ignore 名前

指定したプレイヤーからのメッセージを非表示にします。

/unignore 名前

ignoreしたプレイヤーを解除します。

```
```

便利機能

/ec

エンダーチェストを開きます。

/enderchest

エンダーチェストを開きます。/ec と同じ用途です。

/craft

作業台を開きます。

/hat

手に持っているアイテムを頭に装備します。

/suicide

自分のプレイヤーをリスポーンさせたい場合に使います。アイテムロストに注意してください。

```
```

ギルド

/guild

ギルドGUIを開きます。作成、参加、メンバー確認、クエスト、強化などを行えます。

/guild help

ギルド関連コマンドのヘルプを表示します。

/gc 内容

ギルドに所属しているメンバーだけに見えるチャットを送信します。

```
```

EliteMobs

/em

EliteMobsのメインメニューを開きます。

/em shop

EliteMobs関連のショップを開きます。

/em quests

EliteMobsのクエスト関連メニューを開きます。

/em rank

EliteMobsのランクや進行状況を確認します。

/em wallet

EliteMobs内の通貨や所持状況を確認します。

/em pay 名前 金額

EliteMobsの通貨を他プレイヤーへ送金します。

```
```

Adventurers Guild

/ag

Adventurers Guildのメニューを開きます。

/ag help

Adventurers Guild関連のヘルプを確認します。

/ag menu

Adventurers GuildのGUIメニューを開きます。

/ag rank

Adventurers Guildでのランクや進行状況を確認します。

```
```

保護・確認

/co i

ブロックやチェストなどの履歴確認モードを切り替えます。スタッフ向けです。

/co l

CoreProtectのログを確認します。スタッフ向けです。

/report 名前 理由

問題のあるプレイヤーを運営へ報告します。

```
```

その他

/afk

離席状態を切り替えます。

/ping

自分のpingを確認します。

/realname 名前

ニックネームの元プレイヤー名を確認します。

/nick 名前

ニックネームを設定します。利用可能かどうかはランクや権限によります。

```

注意事項

一部のコマンドは、ランク・権限・ワールド・クールダウン設定によって使用できない場合があります。
コマンドが使えない場合は、運営に確認してください。
